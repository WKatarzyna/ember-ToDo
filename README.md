# emtasks

Ember ToDo application.

## Prerequisites

* [Node.js](https://nodejs.org/) (with npm)
* [Ember CLI](https://ember-cli.com/)


## Installation

* `git clone <repository-url>` this repository
* `cd emtasks`
* `npm install`

## Running / Development

* `ember serve`
*  app avilable at [http://localhost:4200](http://localhost:4200).
